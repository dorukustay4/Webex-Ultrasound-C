Getting Started

The Ultrasound Annotation System is a web-based platform designed for medical professionals to annotate ultrasound images used in ultrasound-guided regional anesthesia (UGRA) procedures. It helps create expert-validated training datasets for AI models.
To start an annotation session:
  1. Click the "Start Session" button on the home page
  2. Upload your ultrasound images
  3. Fill in the session details (doctor name, category)
  4. Begin annotating your images using the annotation tools
The system supports common image formats including:
  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • DICOM files (.dcm)
  • TIFF (.tiff, .tif)

Annotation Process

To annotate nerve structures:
  1. Use the polygon tool to trace around nerve boundaries
  2. Click to place points around the nerve structure
  3. Double-click or press Enter to complete the polygon
  4. Fill in the medical attributes (nerve type, visibility, etc.)
  5. Click "Save Current Annotation"
For each annotation, specify:
  • Nerve Type: Median, Ulnar, Radial, etc.
  • Side of Body: Left or Right
  • Patient Position: Supine, Prone, Lateral
  • Visibility: Poor, Fair, Good, Excellent
  • Age Group: Pediatric, Adult, Geriatric
  • Needle Approach: In-plane, Out-of-plane
  • Clinical Notes: Additional observations
Yes, you can edit annotations within the current session. Use the "Clear All" button to remove all annotations on the current image and start over. Once a session is ended, annotations are saved permanently to the database.

Data Management

You can export annotations in two ways:
  • During session: Click "Export Session Data" before ending session
  • From library: Go to Library page and click "Export" for any completed session
Exports include both annotation data and associated image data in JSON format.
All annotations are stored locally in a secure SQLite database on your computer. This ensures:
  • Data privacy and security
  • Offline access to your work
  • Fast performance
  • No dependency on internet connection
Yes, you can delete sessions from the Library page. Click the "Delete" button for any session. This will permanently remove the session and all associated annotations from the database. This action cannot be undone.

Technical Support

If the application won't start:
  1. Check that you have sufficient system permissions
  2. Ensure no antivirus software is blocking the application
  3. Try running as administrator (Windows) or with sudo (Mac/Linux)
  4. Check the application logs for error messages
  5. Contact technical support if the issue persists
To report bugs or suggest features:
  • Use the Help page to find contact information
  • Include detailed steps to reproduce any issues
  • Attach screenshots if applicable
  • Specify your operating system and application version
