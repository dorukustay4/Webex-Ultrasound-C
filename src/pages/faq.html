Getting Started

The Neurotate System is a comprehensive desktop application designed specifically for medical professionals to annotate ultrasound images used in ultrasound-guided regional anesthesia (UGRA) procedures. It enables doctors to precisely mark nerves boundaries to create expert-validated training datasets for AI model development. The system combines advanced annotation tools with structured medical metadata capture to ensure high-quality, clinically relevant annotations.
No additional software installation is required. The system is a standalone desktop application that includes everything you need. Simply run the application file and you'll have access to all annotation tools, database functionality, and export features. The system works offline and doesn't require an internet connection for core functionality.
Yes, your data is completely secure. All ultrasound images and annotations are stored locally on your computer using an encrypted SQLite database. No data is transmitted to external servers or cloud services. You have full control over your data, including the ability to export, backup, or permanently delete sessions as needed. This ensures complete patient privacy protection.
Starting an annotation session is simple:
  1. From the main dashboard, click the "Start Session" button
  2. Upload your ultrasound images by clicking "Upload Image".
  3. Enter session details including session title, doctor name, and session type.
  4. Click "Begin Annotation" to enter the annotation workspace
The system will automatically create a new session and save your progress as you work.
The system supports a wide range of medical image formats:
  • DICOM files (.dcm) - Industry standard for medical imaging
  • JPEG (.jpg, .jpeg) - Common compressed format
  • PNG (.png) - Lossless compression, excellent for detailed images
Yes! The system supports multiple uploads for efficiency:
  • Add new image files using the "Add Image" button to move on to next annotation after saving first annotation.
All images in a session share the same metadata (doctor, session type, etc.) but can have individual annotations and attributes.

Annotation Process

Follow these detailed steps to annotate nerve structures:
  1. Trace the Nerve Boundary: Click around the perimeter of the nerve structure. Each click places a point
  2. Complete the Shape: Double-click the final point or press Enter to close the polygon
  3. Fill Medical Attributes: Use the form section on the right to specify:
    • Nerve type (Median, Ulnar, Radial, Sciatic, etc.)
    • Side of body (Left/Right)
    • Visibility quality (Poor/Fair/Good/Excellent)
    • Patient position
    • Patient age group
    • Needle approach
    • Clinical notes (if applicable)
  4. Save the Annotation: Click "Save Current Annotation" or Ctrl + S to store the data
Each annotation should include comprehensive medical metadata:
  • Anatomical Structure: Specify the exact nerve (e.g., Median Nerve, Brachial Plexus)
  • Body Side: Left or Right laterality
  • Patient Demographics: Age group (Pediatric, Adult, Elderly)
  • Image Quality: Visibility rating (Poor, Fair, Good, Excellent)
  • Patient Position: Supine, Prone, Lateral, Sitting
  • Needle Approach: In-plane, Out-of-plane (if applicable)
  • Clinical Notes: Any relevant observations, pathology, or anatomical variants
These attributes are crucial for creating clinically relevant training datasets.
Yes, you have full control over your annotations:
  • Modify Attributes: Right-click on an annotation to edit its medical attributes
  • Delete Individual: Select an annotation and press Delete key or right-click → Delete
  • Clear All: Use "Clear All" button to remove all annotations from the current image
  • Undo/Redo: Use Ctrl+Z (undo) and Ctrl+Y (redo) for recent changes
Changes are automatically saved within the current session. Once you end a session, annotations become permanent in the database. Sessions can be viewed, deleted or exported as a whole.

  • Clinical Notes: Additional observations

Data Management

The system provides flexible export options:
  • During Session: Click "Export Session Data" before ending the session
  • From Library: Navigate to Library page and click "Export" for any completed session
Export Format:
  • JSON: Complete annotation data with coordinates and medical attributes.
All exports include both annotation geometry and associated image data (base64 encoded).
Your data is stored securely on your local computer:
  • Database Location: [Application Folder]/data/sessions.db
  • Image Storage: Original images stored as base64 in database for integrity
  • Metadata: All medical attributes and session information included
Backup Strategies:
  • Automatic: Copy the entire application folder to external drive
  • Manual: Export individual sessions as JSON files
  • Professional: Database backup tools can backup the SQLite file
Individual Deletion: Delete specific sessions from Library page Important: Deletion is permanent and cannot be undone. Always export valuable data before deletion.
Advanced search and filtering capabilities:
  • Doctor Name: Find all sessions by specific practitioners
  • Session Title: Find all sessions by specific titles
  • Procedure Type: Filter by session category
  • Date of Session: Filters by specific date ranges
Use the filter panel and search bar in the Library page to combine multiple criteria for precise searches.
