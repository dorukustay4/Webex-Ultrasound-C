Complete guide to using the Ultrasound Annotation System

Quick Start Guide

Follow these steps to get started with annotating ultrasound images:

  1. 1
    Start a New Session
    Navigate to the Home page and click "Start Session". This will initialize a new annotation session and take you to the annotation interface.
  2. 2
    Upload Images
    Click the "Upload Images" button to select your ultrasound images. You can upload multiple images at once.
  3. 3
    Configure Session Details
    Fill in the doctor name and select the appropriate category for your annotation session.
  4. 4
    Start Annotating
    Use the annotation tools to mark nerve structures and other anatomical features in your images.
  5. 5
    Save and Export
    Save your annotations and export the session data when complete.

Annotation Tools

Polygon Tool

The primary tool for outlining nerve structures and anatomical features.

  • Click to place points around the structure
  • Double-click or press Enter to complete
  • Right-click to cancel current polygon

Pan & Zoom

Navigate around the image for detailed annotation work.

  • Mouse wheel to zoom in/out
  • Click and drag to pan
  • Double-click to reset view

Selection Tool

Select and modify existing annotations.

  • Click on annotations to select
  • Drag corner handles to resize
  • Press Delete to remove selected

Medical Attributes

Specify clinical details for each annotation.

  • Nerve type and location
  • Visibility assessment
  • Patient positioning
  • Clinical observations

Best Practices

Annotation Quality

  • Follow nerve boundaries precisely
  • Use adequate zoom for detail work
  • Double-check polygon closure
  • Verify medical attributes

Session Management

  • Save frequently during long sessions
  • Use descriptive session names
  • Export data before ending sessions
  • Review annotations before saving

Data Organization

  • Group related images in sessions
  • Use consistent naming conventions
  • Include relevant clinical notes
  • Backup exported data regularly

Quality Assurance

  • Review all annotations before saving
  • Verify medical attribute accuracy
  • Check for incomplete polygons
  • Validate export data format
💡 Tip: Use the Library page to review and manage your completed annotation sessions. You can view detailed information about each session and export data as needed.
⚠️ Important: Always export your session data before deleting sessions from the library. Deleted sessions cannot be recovered.
